# AgentAudit Skill

Skill này dùng để kiểm tra trạng thái hiện thực của các agent trong thư mục `agents/`.

## Chức năng

- quét toàn bộ agent top-level
- phân loại file thành `implemented`, `partial`, `placeholder`
- đọc export từ `agents/__init__.py`
- phát hiện `missing_exports` và `orphan_exports`
- xuất markdown summary nếu cần

The standalone `agent_audit` CLI was removed from this repository. Keep or refresh `AGENT_AUDIT.md` manually if needed.
